## FinalAssessment: အပြီးသတ် စစ်ဆေးခြင်းနှင့် Mini Project များ

သင်တန်းတစ်လျှောက်လုံး လေ့လာခဲ့တဲ့ Knowledge တွေကို လက်တွေ့အသုံးချပြီး စစ်ဆေးဖို့အတွက် အခု Mini Project အဆင့်ကို ရောက်ရှိလာပါပြီ။

### Final Mini Project (ရွေးချယ်စရာ ပရောဂျက်များ)

သင်တန်းသားများအနေဖြင့် သင်ယူခဲ့သော `pathlib`, `os`, နှင့် `shutil` module များ၏ စွမ်းရည်များကို အပြည့်အဝ အသုံးပြု၍ အောက်ပါ Project (၃) ခုထဲမှ **မိမိနှစ်သက်ရာ တစ်ခု** ကို ရွေးချယ်ပြီး ပြီးဆုံးကြောင်း ပြသရပါမည်။

Project တစ်ခုချင်းစီဟာ သင်ခန်းစာထဲက အဓိက Concept တွေကို အသုံးချဖို့ ရည်ရွယ်ထားတာ ဖြစ်ပါတယ်။

### Project A: Log File Analyzer (Metadata နှင့် ရှာဖွေမှု)

#### ရည်ရွယ်ချက်
သတ်မှတ်ထားသော Log Folder တစ်ခု (ဥပမာ- `logs/server`) အတွင်းရှိ `.log` ဖိုင်များအားလုံးကို Recursive (အဆင့်ဆင့်) ရှာဖွေပြီး၊ ၎င်းတို့၏ စုစုပေါင်း အရွယ်အစားကို တွက်ချက်ပါ။ ထို့အပြင် နောက်ဆုံး ဖန်တီးခဲ့သော Log ဖိုင်၏ အမည်နှင့် Path ကို ဖော်ပြပါ။

#### အဓိက အသုံးပြုရမည့် Concept များ
*   **`pathlib.Path.rglob()`:** Log ဖိုင်များအားလုံးကို နက်ရှိုင်းစွာ ရှာဖွေရန်။
*   **`pathlib.Path.stat()`:** ဖိုင်အရွယ်အစား (`st_size`) နှင့် ဖန်တီးသည့်အချိန် (`st_mtime`) ကို ရယူရန်။
*   **`datetime` Module:** Timestamp များကို နှိုင်းယှဉ်ပြီး နောက်ဆုံးဖိုင်ကို ဆုံးဖြတ်ရန်။

#### လမ်းညွှန်ချက် အတိုချုပ်
1.  Source Directory ကို `Path` Object အဖြစ် သတ်မှတ်ပါ။
2.  `rglob("*.log")` ဖြင့် ဖိုင်များအားလုံးကို စာရင်းပြုစုပါ။
3.  ဖိုင်တစ်ခုစီ၏ အရွယ်အစားကို ပေါင်းပြီး၊ ၎င်း၏ Modification Time ကို အချိန်တိုင်း စစ်ဆေးနှိုင်းယှဉ်ပါ။
4.  ရလဒ်များကို (စုစုပေါင်း အရွယ်အစားနှင့် နောက်ဆုံးဖိုင်အမည်) ပြသပါ။

### Project B: Batch File Renamer (Name and Suffix Manipulation)

#### ရည်ရွယ်ချက်
Folder တစ်ခုအတွင်းရှိ Image ဖိုင်များအားလုံး (ဥပမာ- `.jpg`, `.png`, `.gif`) ကို ရှာဖွေပြီး၊ ၎င်းတို့၏ အမည်များကို **"Photo\_001", "Photo\_002", "Photo\_003"** စသည်ဖြင့် အစဉ်လိုက် (Sequence Number) ပေး၍ အမည်ပြောင်းလဲပေးသည့် Script ရေးသားပါ။

#### အဓိက အသုံးပြုရမည့် Concept များ
*   **`pathlib.Path.iterdir()`:** Folder အတွင်းရှိ Item များကို လှည့်ပတ်ရှာဖွေရန်။
*   **`pathlib.Path.suffix`:** ဖိုင်၏ Extension ကို စစ်ဆေးရန် (Image ဟုတ်/မဟုတ်)။
*   **`pathlib.Path.rename()`:** ဖိုင်အမည်များကို စနစ်တကျ ပြောင်းလဲရန်။
*   **String Formatting:** `"{:03d}".format(number)` ကို အသုံးပြု၍ ဂဏန်းများကို (001, 002) ပုံစံဖြင့် ဖော်မတ်ချရန်။

#### လမ်းညွှန်ချက် အတိုချုပ်
1.  Target Directory ကို သတ်မှတ်ပါ။
2.  ဖိုင်များကို စစ်ဆေးပြီး Image Files များကိုသာ List ထဲသို့ ထည့်ပါ။
3.  `enumerate` ကို အသုံးပြု၍ 0 မှစတင်သော Index ကို ရယူပါ။
4.  Index ကို 001, 002 စသည်ဖြင့် String အဖြစ် ပြောင်းပြီး Path အသစ်ကို ဖန်တီးကာ `rename()` လုပ်ပါ။

### Project C: Directory Comparison Tool (Exists Check and Listing)

#### ရည်ရွယ်ချက်
ပေးထားသော Folder နှစ်ခု (Source A နှင့် Target B) ကို နှိုင်းယှဉ်ပါ။ **Folder A ထဲမှာရှိပြီး Folder B ထဲမှာ မရှိသေးသော** ဖိုင်များစာရင်းကိုသာ ထုတ်ပေးပါ။ ဤစာရင်းသည် Backup ပြုလုပ်ရန် လိုအပ်သည့် ဖိုင်များ ဖြစ်သည်။

#### အဓိက အသုံးပြုရမည့် Concept များ
*   **`pathlib.Path.iterdir()`:** Folder နှစ်ခုလုံး၏ ဖိုင်စာရင်းများကို ရယူရန်။
*   **Python Set Operations:** ဖိုင်အမည်များ၏ List နှစ်ခုကို `set()` အဖြစ် ပြောင်းလဲပြီး Set Difference (`A - B`) ကို တွက်ချက်ကာ မတူညီသော ဖိုင်များကို ရှာဖွေရန်။
*   **`path.is_file()`:** Directory များကို စစ်ဆေးမှုမှ ဖယ်ထုတ်ရန်။

#### လမ်းညွှန်ချက် အတိုချုပ်
1.  Source A နှင့် Target B Folder Path များကို သတ်မှတ်ပါ။
2.  A နှင့် B ထဲက ဖိုင်အမည်များ (Name String) ကိုသာ သီးခြား List နှစ်ခုအဖြစ် ထုတ်ယူပါ။
3.  List များကို Set အဖြစ်ပြောင်းပြီး (Set A - Set B) ကို တွက်ချက်ပါ။
4.  ရလဒ်ထွက်လာသော ဖိုင်အမည်များသည် Target B ထဲတွင် မရှိသေးသော ဖိုင်များ ဖြစ်သည်။
