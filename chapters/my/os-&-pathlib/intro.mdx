## Intro: သင်တန်း မိတ်ဆက် (Welcome to File System Mastery)

မင်္ဂလာပါခင်ဗျာ။ Python File System Mastery သင်တန်းကို တက်ရောက်လာတဲ့အတွက် အထူးပဲ ကြိုဆိုပါတယ်။

ကျွန်တော်တို့ Python Code တွေ ရေးတဲ့အခါ အမြဲတမ်း ကြုံတွေ့ရမယ့် အခြေခံအကျဆုံး အလုပ်တစ်ခုကတော့ **ဖိုင်တွေနဲ့ ဖိုဒါတွေကို စီမံခန့်ခွဲခြင်း** ပဲ ဖြစ်ပါတယ်။ နေ့စဉ်သုံး Data တွေကို Organize လုပ်တာ၊ Log တွေကို စစ်ဆေးတာ၊ ဒါမှမဟုတ် Web Application တစ်ခုအတွက် Files တွေ သိမ်းဆည်းတာပဲဖြစ်ဖြစ်၊ ဖိုင်စနစ်ကို နားလည်ထားဖို့က မရှိမဖြစ်ပါပဲ။

ဒီသင်တန်းရဲ့ ရည်ရွယ်ချက်ကတော့ Python ရဲ့ အခြေခံအကျဆုံးဖြစ်တဲ့ **`os`** နဲ့ ခေတ်မီတဲ့ **`pathlib`** Library တွေကို အားပြုပြီး **Cross-Platform File Management Script** တွေကို ရေးသားနိုင်ဖို့ပဲ ဖြစ်ပါတယ်။

### သင်တန်း၏ အဓိက အာရုံစိုက်ရာများ (Core Focus)

ဒီသင်တန်းတစ်ခုလုံးမှာ ကျွန်တော်တို့ဟာ လက်တွေ့ကျတဲ့ **"Desktop Cleanup Tool"** Project ကို အကောင်အထည်ဖော်ရင်း အဆင့် (၆) ဆင့်ကို ဖြတ်သန်းသွားမှာ ဖြစ်ပါတယ်။

| Module | အဓိက အကြောင်းအရာ | ရည်ရွယ်ချက် |
| :--- | :--- | :--- |
| **Module 1** | နိဒါန်းနှင့် လမ်းကြောင်းများ (Paths) | File System အလုပ်လုပ်ပုံ၊ Absolute နှင့် Relative Path များကို နားလည်ခြင်း။ |
| **Module 2** | `os` Module ဖြင့် အခြေခံများ | သမားရိုးကျ `os` ကိုသုံး၍ Folder များ ဖန်တီးပုံနှင့် Path များကို စနစ်တကျ ပေါင်းစပ်ပုံ (`os.path.join()`)။ |
| **Module 3** | `pathlib` ဖြင့် ခေတ်မီချဉ်းကပ်မှု | `pathlib.Path` Object ကို အသုံးပြု၍ Path များကို ပိုမိုရှင်းလင်းစွာ ကိုင်တွယ်ပုံ။ |
| **Module 4** | ဖိုင်များ စာရင်းပြုစုခြင်း | Folder အတွင်းရှိ ဖိုင်များ၊ Sub-folder များကို `iterdir()` နှင့် `rglob()` ဖြင့် ရှာဖွေနည်း။ |
| **Module 5** | ဖိုင်များ ပြောင်းလဲခြင်းနှင့် MetaData | ဖိုင်များ အမည်ပြောင်းခြင်း၊ ရွှေ့ပြောင်းခြင်း၊ ကူးယူခြင်းနှင့် ဖိုင်၏ Metadata များကို ရယူနည်း။ |
| **Module 6** | Project အကောင်အထည်ဖော်ခြင်း | သင်ခန်းစာအားလုံးကို ပေါင်းစပ်ပြီး "Folder Cleanup Tool" Script ကို အပြီးသတ် ရေးသားခြင်း။ |

### သင်တန်းပြီးဆုံးပါက ဘာတွေရရှိနိုင်မလဲ။ (What to Expect)

ဒီ Course ကို သေချာလိုက်နာမယ်ဆိုရင်၊ သင်ဟာ Python ရဲ့ File System နဲ့ ပတ်သက်တဲ့ **Foundation Level** မှာ လုံးဝ ကျွမ်းကျင်သွားမှာပါ။

*   ✔️ ဘယ် Operating System ပေါ်မှာပဲဖြစ်ဖြစ် အလုပ်လုပ်နိုင်တဲ့ File Management Script တွေကို ကိုယ်တိုင်ရေးသားနိုင်ပါမယ်။
*   ✔️ ဖိုင်တွေကို ရှာဖွေ၊ ဖန်တီး၊ အမည်ပြောင်း၊ ရွှေ့ပြောင်းတဲ့ အလုပ်တွေကို စိတ်ချလက်ချ လုပ်ဆောင်နိုင်ပါမယ်။
*   ✔️ `os` လို သမားရိုးကျ နည်းလမ်းတွေကနေ `pathlib` လို ခေတ်မီတဲ့ OOP နည်းလမ်းကို ဘယ်လိုကူးပြောင်းသုံးစွဲရမယ်ဆိုတာကို သိရှိသွားပါမယ်။
*   ✔️ **Tutorial Hell ထဲကနေ လွတ်မြောက်ပြီး**၊ ကိုယ်ပိုင် Logic နဲ့ Project တစ်ခုကို အစအဆုံး အကောင်အထည်ဖော်နိုင်မယ့် လုပ်ငန်းစဉ်ကို နားလည်သွားပါမယ်။

### ဘာတွေကိုတော့ မမျှော်လင့်သင့်ဘူးလဲ။ (Setting Realistic Limits)

ပွင့်ပွင့်လင်းလင်း ပြောရရင်၊ ဒီ Course ဟာ ဖိုင်စနစ်ကို ကိုင်တွယ်တဲ့ အခြေခံတွေကို အာရုံစိုက်ထားတဲ့အတွက် အောက်ပါ နယ်ပယ်တွေကိုတော့ နက်နက်နဲနဲ သင်ကြားပေးနိုင်မှာ မဟုတ်ပါဘူး။

*   ❌ **Data I/O Streams အသေးစိတ်:** ဖိုင်အများအပြားကို အလွန်လျင်မြန်စွာ ဖတ်ရှုရေးသားခြင်း (Fast File I/O) နည်းစနစ်များ သို့မဟုတ် File Encoding (UTF-8, ASCII) အကြောင်းကို အသေးစိတ် မရှင်းပြထားပါ။
*   ❌ **Advanced Data Libraries:** CSV, JSON, YAML စတဲ့ Data တွေကို စနစ်တကျ ကိုင်တွယ်တဲ့ Pandas, ConfigParser စတဲ့ Library တွေကို မသင်ကြားရသေးပါ။ (ယင်းသည် နောက်ဆက်တွဲ သင်ခန်းစာများအတွက် ဖြစ်ပါတယ်)။
*   ❌ **Networking/Multiprocessing:** Network ပေါ်က ဖိုင်များကို ဝင်ရောက်ကြည့်ရှုခြင်း သို့မဟုတ် ဖိုင်အများအပြားကို တစ်ပြိုင်နက်တည်း လုပ်ဆောင်တဲ့ အဆင့်မြင့် နည်းပညာများကို မဖော်ပြထားပါ။

ဒါပေမဲ့ စိတ်မပူပါနဲ့။ ဒီ Foundation ကို ခိုင်မာအောင် တည်ဆောက်ပြီးတာနဲ့ အဲဒီ Advanced နယ်ပယ်တွေဆီကို အလွယ်တကူ ကူးပြောင်းနိုင်မှာပါ။

ကဲ.. အဆင်သင့်ဖြစ်ပြီဆိုရင်တော့ Module 1 နဲ့ စတင်ပြီး Path တွေရဲ့ လျှို့ဝှက်ချက်တွေကို လေ့လာလိုက်ရအောင်!
